Directivas

En Angular, una directiva es una instrucción que le dice al template cómo debe comportarse o renderizarse un elemento.

Directiva @if ejemplo:

@if (isLoggedIn) {

Bienvenido, {{ username }} 🎉

} @else {

Por favor inicia sesión 🔑

}

Directiva @for ejemplo:

    @for (fruta of frutas; track fruta) {
  • {{ fruta }}
    • }

Directiva @switch ejemplo:

@switch (dia) { @case ('Lunes') {

Hoy es lunes 😴

} @case ('Viernes') {

¡Viernes! 🎉

} @default {

Es un día cualquiera 📅

} }
